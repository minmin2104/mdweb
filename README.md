a static web pages.

## Installation

**REQUIREMENTS**
- [Python](https://www.python.org/downloads/)
- [pipx](https://pipx.pypa.io/stable/installation/)

1. Open your terminal or command line
2. Run
```
pipx install git+https://github.com/minmin2104/mdweb.git
```

After that you can use the tool

Run 
```
mdweb -h
```
to get started

## Contribution

**REQUIREMENTS**
- [Python](https://www.python.org/downloads/)
- [pipx](https://pipx.pypa.io/stable/installation/)

1. Fork the repo
2. Clone to your computer
3. Open your terminal or command line
4. Run
```
pipx install --editable .
```
