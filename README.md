# MD Web

This tools allow you to write your blog in Markdown Files and convert it into
a static web pages.

> [!IMPORTANT]
> ## Early access
> This tool is still in it development state but already is usable with a few
> limitation. Take a look at
> [Issues](https://github.com/minmin2104/mdweb/issues). If you've found a bug
> that you'd like to be fixed, open an issue
> [here](https://github.com/minmin2104/mdweb/issues).
